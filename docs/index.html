Usage
Two steps. Infinite flexibility.
Full Suite — Jenkinsfile
def report = forgeAI(
  analyzers: [
    'code-review',
    'vulnerability',
    'architecture-drift',
    'test-gaps',
    'release-readiness'
  ],
  sourceGlob: 'src/**/*.java',
  contextInfo: 'Spring Boot, hexagonal arch',
  failOnCritical: true,
  criticalThreshold: 4
)
echo "Score: ${report.compositeScore}"
Targeted Scan — Single Analyzer
def sec = forgeAIScan(
  analyzer: 'vulnerability',
  source: readFile('src/App.java'),
  context: 'Java 17 REST API, PII data'
)

if (sec.criticalCount > 0) {
  error("${sec.criticalCount} critical vulns")

Quick start
Up and running in 5 minutes
1

Install the plugin

Download the .hpi from GitHub Releases and upload via Manage Jenkins → Plugins → Advanced.

mvn clean package -DskipTests
2

Configure your LLM

Go to Manage Jenkins → System → ForgeAI. Pick OpenAI, Claude, Groq, or Ollama. Hit Test Connection.

Provider: Groq
Model: llama-3.3-70b-versatile
3

Add to your pipeline

One step in your Jenkinsfile. Eight dimensions of intelligence. A report on every build.

forgeAI analyzers: ['code-review',
'vulnerability', 'architecture-drift']
